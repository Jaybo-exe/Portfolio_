## Getting Started

Simply open `index.html` in your web browser to view the portfolio.

### Customization

1. Open `index.html` and replace placeholder text:
   - Your name in the hero section
   - About section content
   - Skills and descriptions
   - Project information
   - Contact links (email, GitHub, LinkedIn)

2. Optional CSS customization in `styles.css`:
   - Colors (search for color codes like `#2563eb`)
   - Fonts (change `font-family` values)
   - Spacing and sizes
